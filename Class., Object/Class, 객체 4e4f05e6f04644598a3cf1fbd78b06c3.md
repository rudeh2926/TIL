# Class, 객체

---

### 객체

**(객체)현실 세계에 있는 모든 것**  물리적과 추상적으로 나눔

1. 식별 가능
2. 각자의 속성이 있음 (정보) 

**속성 = 필드 = 클래스 내의 변수**

---

### Class

자바에서 **설계도는 클래스(class**)이다.

클래스에는 객체를 생성하기 위한 **필드(정보)와 메소드가 정의**되어 있다.

클래스로부터 만들어진 객체를 해당 클래스의 **인스턴트(Instance)**라고 한다.
